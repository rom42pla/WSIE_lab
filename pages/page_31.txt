            <h1><a name="Economia_Aziendale_AA_dal_2004_a"></a> Economia Aziendale (AA dal 2004 al 2011) </h1> Docente: Romano Boni - Assistente: Silvia Torrani 
            <p></p> 
            <h2><a name="PROGRAMMA_DEL_CORSO"></a> PROGRAMMA DEL CORSO </h2> 
            <p></p> 
            <ol> 
             <li> Definizioni e temi da affrontare: 
              <ul> 
               <li> l&#x2019;azienda, l&#x2019;impresa, l&#x2019;imprenditore; </li> 
               <li> organizzazione e gestione dell&#x2019;impresa; </li> 
               <li> capitale d&#x2019;impresa: rischio e finanziamento. </li>
              </ul> </li> 
             <li> Il quadro di insieme: 
              <ul> 
               <li> gli scenari economici: i settori d&#x2019;impresa </li> 
               <li> l&#x2019;evoluzione della economia italiana; </li> 
               <li> la caratteristiche delle imprese italiane; </li> 
               <li> le aziende italiane oggi: minacce ed opportunità; </li> 
               <li> le maggiori trasformazioni in corso; </li> 
               <li> l&#x2019;evoluzione della economia mondiale; </li> 
               <li> le crisi di fine millennio; </li> 
               <li> la globalizzazione dei mercati: un approfondimento. </li>
              </ul> </li> 
             <li> L&#x2019;organizzazione della azienda: 
              <ul> 
               <li> le forme societarie nel nostro ordinamento giuridico; </li> 
               <li> l&#x2019;organizzazione di governo dell&#x2019;azienda; </li> 
               <li> l&#x2019;evoluzione delle strutture organizzative tradizionali; </li> 
               <li> la crescita del fattore conoscenza; </li> 
               <li> i sistemi, i processi, la cultura . </li>
              </ul> </li> 
             <li> La gestione aziendale: 
              <ul> 
               <li> la pianificazione strategica: missione, obiettivi, processo di planning; </li> 
               <li> gli investimenti: valutazione e controllo; </li> 
               <li> la progettazione dei prodotti: dal mercato alla produzione </li> 
               <li> il marketing: e l&#x2019;attività commerciale; </li> 
               <li> la gestione della produzione: i metodi occidentale (MRP) e giapponese (JIT); </li> 
               <li> il bilancio economico, la contabilità, la finanza, il controllo di gestione; </li> 
               <li> il bilancio etico e il bilancio ecologico; </li> 
               <li> la gestione e lo sviluppo delle risorse umane; </li> 
               <li> il sistema informativo: supporto alla gestione aziendale. </li>
              </ul> </li> 
             <li> La reingegnerizzazione dei processi e le nuove tecnologie dell'informazione: 
              <ul> 
               <li> modelli funzionali ed informativi nei processi aziendali; </li> 
               <li> la reingegnerizzazione dei processi tramite informatica e TLC; </li> 
               <li> la gestione del cambiamento; </li> 
               <li> i nuovi processi: telelavoro, RFId, CRM, VOIP, portali intranet, e-invoicing, aste online, e-learning,etc. </li>
              </ul> </li>
            </ol>

var twTopMenuBarCloseTimer = null;
var twTopMenuBarTimerMsec = 1000;
function twToggleTopMenuBar( isHover ) {
  if( 'off' == 'on' || 'off' == '1' ) {
    return;
  }
  var displayState = 'none';
  if( $(window).scrollTop()<22 || isHover>0 ) {
     displayState = 'block';
     if(twTopMenuBarCloseTimer) { window.clearTimeout(twTopMenuBarCloseTimer); twTopMenuBarCloseTimer = null; } 
  } else {
    if(!twTopMenuBarCloseTimer && isHover>=0 ) {
      twTopMenuBarCloseTimer = window.setTimeout(twCloseTopMenuBar, twTopMenuBarTimerMsec);
      return;
    }
  }
  if( $('#topmenu-bar').css('display') != displayState ) {
    $('#topmenu-bar').css('display', displayState);
  }
};
function twCloseTopMenuBar() {
  twToggleTopMenuBar(-1);
}
$(document).ready(function () {
  twToggleTopMenuBar(0);
});
$(window).scroll(function () {  
  twToggleTopMenuBar(0);
});

  function () { twToggleTopMenuBar(1); },

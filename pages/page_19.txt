            <h2><a name="Course_Organization"></a> Course Organization </h2> 
            <p></p> The course presents architectures and algorithms related with the extraction of information from the Web, analyzing both Web Search engines and on-line Social Networks. A number of lessons are held in the Colossus lab. 
            <p></p> During the lab, students will learn: 
            <ul> 
             <li> Lucene, an open-source text-search library. With Lucene, you will learn to index and search a document archive </li> 
             <li> How to useTwitter API, to track and index Twitter messages, to create and analyze word time series, and more </li> 
             <li> To build a web scraper and trace the content of a forum </li>
            </ul> 
            <p></p> 
            <h2><a name="Self_Assessment_and_Final_Projec"></a> Self Assessment and Final Project </h2> 
            <p></p> Self assessments and final project are sent to all students by email (google group). 
            <h2><a name="Summary_of_Course_Topics"></a> Summary of Course Topics </h2> 
            <p></p> 
            <strong>Architecture of Information Retrieval systems</strong>. Tokenisation, stop-word removal and stemming; morphology; selection of index terms, use of thesauri. Inverted indices. Boolean and vector-space retrieval models, ranked retrieval and text-similarity metrics. Performance metrics: recall, precision, and F-measure. Evaluations on benchmark text collections. Latent Semantic Indexing. Relevance feedback. Query expansion. 
            <p></p> 
            <strong>Web Information Retrieval</strong>. Browsing and Scraping. Link analysis: Page Rank and HITS. 
            <p></p> 
            <strong>Social Network analysis</strong>: Opinion Mining, Social Network analysis, Community Detection, Social Media Analytics, Recommender systems 
            <p></p>

            <h2><a name="Syllabus"></a> Syllabus </h2> 
            <h2><a name="Part_A_Web_Information_Retrieval"></a> Part A: Web Information Retrieval </h2> 
            <ul> 
             <li> Introduction, Architecture of IR systems </li> 
             <li> Text processing, Indexing </li> 
             <li> Boolean and Vector Space Models </li> 
             <li> Query expansion, understanding users' needs </li> 
             <li> Ranking and query expansion based on word similarities (Singular value Decomposition, Word Embeddings) </li> 
             <li> Evaluation methods: experimental and theoretical methods </li> 
             <li> Web Information Retrieval </li> 
             <li> Link Analysis </li>
            </ul> 
            <p></p> 
            <h2><a name="Part_B_Social_Information_Extrac"></a> Part B:Social Information Extraction </h2> 
            <p></p> 
            <ul> 
             <li> Social Network Analysis: graph-based measures, community detection, topic diffusion, temporal analysis </li> 
             <li> Opinion Mining </li> 
             <li> Recommender Systems </li>
            </ul> 
            <p></p> 
            <h2><a name="Topics_for_Final_Dissertation"></a> Topics for Final Dissertation </h2> 
            <p></p> e-health, network medicine, event detection in social media and on the web, enterprise social networks, applications to social studies, temporal information retrieval, semantic recommenders, prediction and edetection of trendy topics 
            <p></p> 
            <strong>Merit-based</strong> (= high grades, very good programming skills) scholarships are available - ask the instructor-

var twTopMenuBarCloseTimer = null;
var twTopMenuBarTimerMsec = 1000;
function twToggleTopMenuBar( isHover ) {
  if( 'off' == 'on' || 'off' == '1' ) {
    return;
  }
  var displayState = 'none';
  if( $(window).scrollTop()<22 || isHover>0 ) {
     displayState = 'block';
     if(twTopMenuBarCloseTimer) { window.clearTimeout(twTopMenuBarCloseTimer); twTopMenuBarCloseTimer = null; } 
  } else {
    if(!twTopMenuBarCloseTimer && isHover>=0 ) {
      twTopMenuBarCloseTimer = window.setTimeout(twCloseTopMenuBar, twTopMenuBarTimerMsec);
      return;
    }
  }
  if( $('#topmenu-bar').css('display') != displayState ) {
    $('#topmenu-bar').css('display', displayState);
  }
};
function twCloseTopMenuBar() {
  twToggleTopMenuBar(-1);
}
$(document).ready(function () {
  twToggleTopMenuBar(0);
});
$(window).scroll(function () {  
  twToggleTopMenuBar(0);
});
$('#topmenu-start').hover(
  function () { twToggleTopMenuBar(1); }, 
  function () { twToggleTopMenuBar(0); }
);
